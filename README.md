A simple, responsive, and highly customizable personal link hub page. This project acts as a digital business card, allowing users to consolidate all their important links (social media, GitHub, portfolios, and contact info) into one single, mobile-friendly landing page.

## 🚀 Features

*   **Mobile-First Design:** Fully responsive and optimized for smartphone screens, just like the real LinkTree.
*   **Clean UI/UX:** Minimalist and modern interface with hover effects and smooth transitions.
*   **Zero Dependencies:** Built entirely with plain HTML and CSS. No heavy frameworks or libraries required.
*   **Easy Deployment:** Ready to be hosted for free via GitHub Pages.

## 🛠️ Usage & Customization

To use this template for your own profile, follow these steps:

1.  **Clone or Download** this repository to your local machine.
2.  Open the `index.html` file in your preferred code editor.
3.  **Update the Avatar:** Replace the `profile-pic.jpg` file in the images folder with your own photo.
4.  **Edit the Bio:** Update the text to reflect your background. 
```html
    <!-- Example -->
    <h1>Your Name</h1>
    <p>CSE Student @ M.Kumarasamy College of Engineering | Java | IoT | DLT</p>
    ```
5.  **Add Your Links:** Modify the `href` attributes in the link buttons to point to your actual social media profiles and projects.
```html
    <a href="[https://github.com/yourusername](https://github.com/yourusername)" class="link-btn">GitHub</a>
    <a href="mailto:your.email@example.com" class="link-btn">Email Me</a>
    ```
6.  **Tweak the Styles:** Open `style.css` to change the background colors, font families, and button shapes to match your personal brand.

## 🌍 How to Deploy (For Free)

You can easily host this page online using **GitHub Pages**:

1.  Create a new repository on GitHub and upload your code.
2.  Go to the repository **Settings**.
3.  On the left sidebar, click on **Pages**.
4.  Under the "Build and deployment" section, select `main` from the **Branch** dropdown menu and click **Save**.
5.  Wait a minute or two, refresh the page, and GitHub will provide you with a live URL (e.g., `https://yourusername.github.io/LinkTree-Clone`).
